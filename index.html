Cara Deface PoC Take over domain menggunakan github

Cara Deface PoC Take over domain menggunakan github

Mr.XxX
January 09, 2021 January 16, 2021

Assalammualaikum wr.wb,balik lagi bersama mimin blog security-xploit7.dalam pertemuan kali ini mimin akan memberikan tutorial deface website metode/proof of concept(poc) Take Over Domain.ok langsung saja mimin kasih caranya.

Bahan tempur yang diperlukan:

1.Akun Github,jika kalian belom punya akun githubnya kalian bisa daftar dulu disini.

2.akun dns afraid,jika belum silahkan buat terlebih dahulu disini

3.script depes kalian format/ekstensinya harus .html

4.kuota yang cukup

5.niat


Langkah-langkah:

1.kalian masuk dulu di akun github kalian.untuk yang tidak tau cara mendaftarnya bisa baca disini untuk tutorial buat akun github

2.jika sudah,kalian login dulu di akun dns afraid kalian disini

3.kalian klik registry,berada diatas menu logout.untuk jelasnya bisa liat gambar dibawah



4.Nah jika sudah,kalian cari nama domain yang mau jadi dijadiin target.caranya kalian ketik dikolom pencariannya letaknya diatas pojok kanan,seperti gambar dibawah ini



5.disitu contoh mimin mencari target malaysia.dan untuk domainnya .my

6.dan jika klik search dan akan keluar banyak domain .my

7.pilih salah satu domain yang menurut kalian enak buat di depes,caranya dengan klik nama domain tersebut.contoh mimin disitu pilih domain ast.my,untuk gambarnya bisa liat di langkah ke 4

8.jika sudah kalian klik nama domainnya,kalian isi subdomainnya sesuai keinginan kalian.contoh disini mimin menggunakan subdo kualalumpur.gambar bisa liat dibawah ini



9.dari gambar diatas mimin belum rubah destinationnya,untuk destinationnya diatas masih default.lalu kalian ganti dengan ip server nya 185.199.110.153.contoh seperti dibawah



10.nah jika sudah kalian ketik kode keamanannya untuk verifikasi.setelah itu klik save.dan jika berhasil ditambahkan tampilannya seperti dibawah ini



11.jika sudah sukses kalian balik ke github lagi untuk create new repository.caranya klik icon plus diatas pojok kanan.pilih new repository.contoh gambar



12.jika sudah pastekan link subdo yang telah kalian buat tadi di dns afraid pada repository name,dan klik create repository

13.setelah kalian create repo,kalian klik creating a new file.contoh gambar:



14.jika sudah namai file dengan nama index.html dan buat script deface kalian dibawah.contoh gambar:



15.jika sudah,kalian klik commit new file

16.jika sudah terbuat file indexnya sekarang kalian klik titik 3 sejajar dan pilih setting.seperti gambar dibawah:



17.jika sudah,kalian scrool sampai Github Pages.dan pada kolom source pilihan defaultnya masih none nah itu kalian ganti jadi main setelah itu klik save.contoh:



18.jika sudah kalian scrool lagi ke bawah sampai Github Pages dan pada kolom domain custom kalian isikan nama domainnya sesuai yang kalian buat tadi di dns afraid

19.kalian buka di tab baru dan pastekan link subdo tadi di tab baru dan boom website has been hacked bai:V.contoh punya mimin:



Nah jadi seperti itulah cara deface poc take over domain menggunakan github,sampai disini tutorial kali ini jika masih belom paham bisa tanya di kolom komentar dan terimakasih sudah mampir untuk membaca🤗
